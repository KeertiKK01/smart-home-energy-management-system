# ⚡ Smart Home Energy Management System

A **Smart Home Energy Management System** that allows users to monitor, manage, and optimize their home energy usage.  
The system enables users to **add devices, track energy consumption, and analyze electricity costs through a dashboard**.

---

# 📌 Features

## 👤 User Authentication
- 🔐 User Registration with OTP verification
- 🔑 Secure Login system
- 🔁 Forgot Password with OTP reset
- 🛡 Protected routes for authenticated users

## 📱 Device Management
- ➕ Add smart home devices
- ⚙️ Enable / Disable devices
- ❌ Remove devices
- 👥 Devices stored **user-wise**

## 📊 Dashboard
- 📟 Total connected devices
- 🟢 Active devices monitoring
- ⚡ Energy consumption calculation
- 💰 Monthly electricity cost
- 📅 Yearly electricity cost

## 👤 Profile Management
- 🧑 View user profile
- 🖼 Upload profile picture
- 🏠 Save and manage address information

## 📈 Energy Analytics
- 📊 Daily energy usage chart
- 📉 Monthly energy usage chart
- 📆 Yearly energy usage chart

# 🚀 Installation & Setup

## 1️⃣ Clone the Repository
git clone https://github.com/yourusername/smart-home-energy-management-system.git

---

## 2️⃣ Backend Setup (Spring Boot)

Navigate to backend folder and run:
mvn spring-boot:run
Backend will start on:
http://localhost:8081


---

## 3️⃣ Frontend Setup (React)

Navigate to frontend folder and run:
npm install
npm start

Frontend will start on:  http://localhost:3001
